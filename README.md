# Laravel Backup Manager

[![codecov](https://codecov.io/gh/SameOldNick/laravel-backup-manager/graph/badge.svg?token=SgfZngm6IB)](https://codecov.io/gh/SameOldNick/laravel-backup-manager)

A flexible Laravel backup package that centralizes backup and cleanup schedules in the database, supports dynamic storage disks, and extends [Spatie Backup](https://github.com/spatie/laravel-backup) for real-world production workflows.
